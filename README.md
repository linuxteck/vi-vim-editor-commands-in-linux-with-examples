If you learn just these 5 commands, you're already ahead:

- `Esc` → exit to normal mode  
- `:w` → save file  
- `:q` → quit  
- `dd` → delete a line

## 🔄 15 Essential vim Commands

| Command | What It Does |
|--------|--------------|
| `i` | Enter insert mode |
| `Esc` | Return to normal mode |
| `:w` | Save file |
| `:q` | Quit editor |
| `:wq` | Save and quit |
| `:q!` | Quit without saving |
| `dd` | Delete current line |
| `yy` | Copy (yank) line |
| `p` | Paste below |
| `u` | Undo last change |
| `Ctrl + r` | Redo change |
| `gg` | Go to first line |
| `G` | Go to last line |
| `/text` | Search forward |
| `n` | Next search result |
